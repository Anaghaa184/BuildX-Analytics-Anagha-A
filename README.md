# Indian Startup Funding Analysis (2015–2025)

## 📊 Project Overview
This project analyzes startup funding trends in India, covering data cleaning, SQL-based business analysis, and an interactive Power BI dashboard. It explores how funding is distributed across industries, cities, and investment types, with the goal of identifying patterns in the Indian startup ecosystem.

## 📁 Dataset Selected
**Indian Startup Funding Dataset**, containing records of startup funding deals including startup name, industry, sub-vertical, city, investors, investment type, and funding amount (in USD).

## 🎯 Why I Selected This Dataset
I chose this dataset because it offers a rich mix of business, financial, and geographical dimensions, allowing for meaningful analysis beyond simple summary statistics. Startup funding data is also highly relevant to real-world business and investment decision-making, making it a strong choice for demonstrating practical data analysis skills relevant to analytics roles.

## ❓ Business Questions Answered
1. Which industries have received the highest total funding in India?
2. Which cities are the leading startup funding hubs?
3. How does funding amount vary across different investment types (Seed, Series A, Private Equity, etc.)?

## 🛠 Tools Used
- **Python (Pandas, NumPy)** — Data cleaning and preprocessing in Google Colab
- **SQL (SQLite)** — Business question analysis using SQLiteOnline
- **Power BI** — Interactive dashboard and visualization
- **GitHub** — Version control and project documentation

## 💡 Key Insight
> Bangalore alone accounted for approximately **30%** of total startup funding in India between 2015–2025, making it the dominant hub for investment activity in the dataset.
>The Ecommerce industry raised the highest cumulative funding at $7.2bn, nearly 1.14 times more than the second-highest industry Consumer internet
>Over 47.41% of recorded deals fall into the 'Undisclosed' funding category, highlighting a transparency gap in Indian startup reporting.
